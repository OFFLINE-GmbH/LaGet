How to use LaGet

LaGet is a NuGet package server. To browse this repository, add {{ urls.api_nuget }} to your NuGet package manager configuration.

To upload packages to this LaGet server download the NuGet command line tool.
Using the command prompt, change to your project's directory.
Type nuget spec to generate a .nuspec file. Edit this file like as wish.
Type nuget pack YourProject.csproj -Prop Configuration=Release to pack your project.
Type nuget push YourProject.nupkg {{ authorizedUser.apikey }} -Source {{ urls.api_upload }} to push your package to the server.
